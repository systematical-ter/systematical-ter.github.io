3/12 BBCF Notes

Published: Wed 12 March 2025
By Systematical

In bbcfnotes.

It was mostly a fine BBCF day at first? TMN was fine.

Heads up that this is a little bit emo so feel free to skip to a section or whatever if you want. Just need to get this off my chest, and if I can't write about it in my personal blog, what the fuck's the point of this blog?

I got some good practice matches in with Augo, and I think I actually really like the Kagura matchup even though I think it's Kagura-favored. It's explosive, and that's kind of fun.

I almost took my first set today. Would have been neat, but I didn't respect Bullet's drives enough.

I could have won my second set but I didn't respect the size of Es' buttons. That bothered me a little because again I lost something I should've won but I turned off my brain and ran into stuff.

EDIT:

I got over it. :) Unironically. Ranting about it helped. Removing a lot of this because I named a lot of people by name and I don't want to get screwed by people finding this accidentally by searching people's names. If there's one thing you need to know about me it's that I will feel things to the max and I do my best to make sure other people don't get affected by it, because I'll get over it. (That's part of why I'm glad I started fighting games so late; if I started earlier, I wouldn't have been so practiced in getting over it and I probably would have crashed out explosively.)

Got lotta games in with my usual sparring buddy (I think I can call him that?) today. He beat my ass as usual, but I got the chance to get some 6a practice in and I took a few matches despite being an emotional and physical wreck.

Notes

6a progress is going not bad. I'm using it now. I'm able to confirm it with both 2D and a jc j.c, but now I need to work on getting the [6]b confirm out.

Still need to work on movement. Copied from yesterday's:

  • Dash barrier block.
  • In the corner: 9 jump with immediate BB.
  • 96 dash. I keep doing 766 which is way way way too predictable.

Also I've lost some combos, somehow.

  • Stop fucking up my 4d j.b j.c jc j.c j.2d combo.
  • Need to get 6c -> 22cccc -> OD -> 66d -> 6321466d consistent again.

I randomly started getting better at bullshitting combos, though. Landed some nice freestyles today, so I might be actually learning how moves go into each other.

Not many other notes today. I'm sure I can find something but this is plenty to work on for now so no reason to. Maybe I'll find something when I go back over my vods.
